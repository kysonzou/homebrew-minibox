# Homebrew Tap for Minibox

This is the official Homebrew tap for [minibox](https://github.com/kyson/minibox).

## Installation

```bash
brew tap kyson/minibox
brew install minibox
```

## Usage

After installation:

```bash
# Check version
minibox version

# Start service
sudo minibox run

# Enable autostart
sudo minibox autostart on
```

## Updating

The formula is automatically updated when a new version is released.

To update manually:

```bash
brew update
brew upgrade minibox
```
